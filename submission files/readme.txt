The data for this project was collected by the United States Forest Service (USFS). The datasets for this project were from https://www.data.gov/ in comma separated values (csv) format. The fire occurrence point data was acquired from 
https://catalog.data.gov/dataset/national-usfs-fire-occurrence-point-feature-layer-d3233
And the fire perimeter data was acquired from
https://catalog.data.gov/dataset/national-usfs-fire-perimeter-feature-layer-bb7cd

The code used to create tables in pgAdmin from the csv files can be found in our GitHub repository at
https://github.com/luca-luna/fire-management
The SQL code in “submission files/create.sql” and “submission files/load.sql” was executed to create tables with the appropriate types and import the data.
